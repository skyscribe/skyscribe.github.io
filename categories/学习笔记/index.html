    1 minute read

    C++11新特性2-RValue Reference 与 Move

    2 minute read

    现有的C++03标准中，不必要的对象的拷贝和临时对象的构造经常会造成额外的性能开销（即使有返回值优化这样的编译器优化来帮忙也不能解决好多情况的问题）；新的C++11标准通过对语言的修正，引入了RValue Reference和Move来解决这一问题。
